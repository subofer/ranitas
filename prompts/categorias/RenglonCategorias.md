## PROPÓSITO GENERAL
Componente de renglón en tabla de categorías mostrando nombre, contador de productos y acciones.

## PROPS
- categoria: object - {id, nombre, _count?: {productos}}
- onEdit: function - Callback para editar
- onDelete: function - Callback para eliminar

## FUNCIONALIDADES

### Display
- Nombre de categoría
- Contador de productos (si existe _count)
- Acciones: Editar, Eliminar

### Botones de acción
- Editar: icono lápiz
- Eliminar: icono trash
