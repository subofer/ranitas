## PROPÓSITO GENERAL
Página de punto de venta (POS) para realizar ventas rápidas.

## FUNCIONALIDADES

### Flujo de Venta
1. Buscar producto (por código/barras o nombre)
2. Agregar a carrito
3. Ajustar cantidades
4. Aplicar descuentos
5. Seleccionar método pago
6. Procesar venta

### Carrito
- Lista de productos agregados
- Cantidad x producto
- Precio unitario
- Total línea
- Botón quitar

### Totalizadores
- Subtotal
- Impuestos (IVA)
- Total
- Cambio (si aplica)

### Métodos de Pago
- Efectivo
- Tarjeta (débito/crédito)
- Cheque
- Transferencia

### Acciones
- Vaciar carrito
- Generar factura/ticket
- Imprimir
- Email recibo
- Guardar venta
