## PROPÓSITO GENERAL
Componente que usa cámara para escanear códigos de barras o QR.

## FUNCIONALIDADES
- Acceso a cámara del dispositivo
- Detección de códigos de barras/QR
- Overlay con guía de escaneo
- Visor en vivo

## PROPS
- onScan: function - Callback al detectar código
- onClose: function - Cierra scanner
- autoFocus?: boolean - Enfoque automático
- mimeType?: string - Tipo de formato

## FEATURES
- Video stream en vivo
- Detección automática
- Manejo de permisos
- Error si no hay cámara
- Flash toggle (si disponible)
- Botón captura manual

## CASOS DE USO
- Scan de productos
- Lectura de códigos de entrada
- Control de inventario
