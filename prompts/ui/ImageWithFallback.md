## PROPÓSITO GENERAL
Componente que carga imagen con fallback a placeholder si falla.

## PROPS
- src: string - URL de imagen
- alt: string - Texto alternativo
- fallback?: ReactNode - Componente fallback
- width?: number - Ancho
- height?: number - Alto
- className?: string

## FUNCIONALIDADES
- Intenta cargar imagen
- Si falla: muestra fallback/placeholder
- Error handling
- Loading state

## ESTILOS
- Objeto fit cover
- Responsive
