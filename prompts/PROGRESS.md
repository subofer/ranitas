## Recomendaciones de Uso

### Para Regenerar un Componente
1. Ir a `/prompts/[categoria]/[componente].md`
2. Copiar contenido del prompt
3. Pegar en Claude/ChatGPT
4. El prompt incluye NUEVAS CARACTERÍSTICAS sugeridas

### Para Crear Nuevos Componentes
1. Revisar prompts similares
2. Usar como template
3. Agregar carpeta si categoría nueva

### Para Actualizar la Documentación
1. Mantener estructura consistent
2. Incluir NUEVAS CARACTERÍSTICAS
3. Actualizar PROGRESS.md con cambios
