Página para importar/exportar datos en formato Excel/CSV.

## FUNCIONALIDADES

### Importar
- Upload archivo Excel/CSV
- Preview de datos
- Validación de estructura
- Mapeo de columnas
- Import confirmado

### Qué importar
- Productos (batch)
- Contactos/clientes
- Categorías
- Proveedores
- Precios

### Exportar
- Seleccionar qué exportar
- Botón descargar Excel
- Formato template
- Incluye validaciones

### Validaciones
- Campos requeridos
- Formatos válidos
- Duplicados
- Errores por fila

### Resultados
- Tabla de resultados
- Elementos exitosos
- Elementos con error (detalle)
- Resumen
