## PROPÓSITO GENERAL
Componente para agregar/remover categorías dinámicamente a un array.

## PROPS
- categorias: array - Array de categorías seleccionadas
- onChange: function - Callback al cambiar array
- placeholder?: string

## FUNCIONALIDADES

### Display
- Tags/chips por cada categoría
- Input para buscar/agregar
- Botón X para remover

### Lógica
- Autocompletar mientras escribe
- Enter para agregar
- No permitir duplicados
- OnChange con array actualizado
