## PROPÓSITO GENERAL
Función que muestra una alerta visual SweetAlert2 preguntando si el usuario está seguro 
de eliminar un producto, con foto grande del producto y acciones (Sí/No).

## PROPS PRINCIPALES (función)
- {imagen, nombre}: object
  - imagen: string (URL de la imagen del producto)
  - nombre: string (nombre del producto)
- action: function - Server action que ejecuta la eliminación

### 1. ALERTA INICIAL (pregunta)

HTML personalizado: Muestra imagen grande del producto
- Imagen: width 320px
- Animación: float 2s ease-in-out infinite
- Centrada horizontalmente

Botones:
- Cancelar: "No, Para un poco...!" (color rojo #d33)
- Confirmar: "Sí, bórralo, soy loco!" (color azul #3085d6)
