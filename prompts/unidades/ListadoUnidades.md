## PROPÓSITO GENERAL

- Columnas: nombre, abreviatura, símbolo, acciones
- Renglones con RenglonUnidades

### Botones
- Agregar nueva unidad
- Editar
- Eliminar (con alerta)

### Estados
- Loading (ProductListPlaceholder)
- Empty (sin unidades)
- Error handling

### Interacción
- Modal para crear/editar
- Alerta para confirmar eliminación
- Refresh list después de cambios
