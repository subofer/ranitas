PROMPT IDEAL PARA REGENERAR IaPromp.jsx

## PROPÓSITO GENERAL
Componente para entrada de prompts a IA con ayudantes y plantillas.

## FUNCIONALIDADES

### Input
- Textarea para prompt
- Placeholder con ejemplos
- Counter caracteres
- Clear button

### Ayudantes
- Botones predefinidos (Resumir, Traducir, etc)
- Plantillas de prompts
- Historial de prompts

### Submit
- Botón enviar
- Loading state
- Error handling

## PROPS
- onSubmit: function - Callback con prompt
- plantillas?: array - Custom plantillas
- maxLength?: number
