## PROPÓSITO GENERAL
Componente esqueleto para loading states (placeholder animado).

## PROPS
- width?: string - Ancho (ej: "w-full", "w-1/2")
- height?: string - Alto (ej: "h-4", "h-12")
- circles?: boolean - Forma circular
- count?: number - Cantidad de skeletos repetidos
- className?: string - Classes adicionales

## FUNCIONALIDADES
- Animación de pulsación
- Fondo gris claro
- Bordes redondeados
- Repetible

## CASOS DE USO
- Placeholder de tablas
- Placeholder de cards
- Skeleton de imágenes
- Skeleton de texto
