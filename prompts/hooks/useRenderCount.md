## PROPÓSITO GENERAL
Hook de debug que cuenta cuántas veces se renderiza un componente.

## API

### useRenderCount(componentName = '')
- Parámetro: componentName - nombre para console log
- Devuelve: count (número de renders)

## FUNCIONALIDADES
- Cuenta renders en componente
- Log en console
- Útil para debug de performance
- Muestra dependencies que cambian
